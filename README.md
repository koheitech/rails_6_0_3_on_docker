# README

This repo is boilerplate for rails v6.0.3 on docker.
You can start off without worring about dependencies and all.

## How to use

### 1: clone this repo

```basg
git clone git@github.com:koheitech/rails_6_0_3_on_docker.git
```

### 2: start container

```bash
docker compose up
```
